# Model
---
Un __modello di Machine Learning__ è una funzione matematica o statistica che apprende un comportamento dai dati per fare previsioni o prendere decisioni senza essere esplicitamente programmato.  Un modello ha il compito di catturare e descrivere le relazioni tra i dati.

Formalmente, un modello di apprendimento automatico può essere visto come una funzione $$ h: X \rightarrow Y $$ dove $X$ è lo spazio delle caratteristiche di input e $Y$ è lo spazio delle etichette o valori di output. 

L'obiettivo del modello è apprendere una buona approssimazione della funzione sconosciuta che collega gli input agli output a partire da un insieme di dati di addestramento.

## Spazio delle ipotesi
---

Con un modello di machine learning si definisce la classe delle funzioni che può rappresentare detta **Spazio delle ipotesi** $H$, che include tutti i set delle funzioni $h(\mathbf{x},\mathbf{w})$.

## Caratteristiche
Le principali caratteristiche di un modello di machine learning sono:
- Esempi di training 
- Funzione target
- Ipotesi $h$
- Spazio delle ipotesi $H$


## Tipi di modello
---
Esistono diversi tipi di modelli tra cui:
- [[Linear models]]
- [[Symbolic rules]]
- [[Probabilistic models]]
- [[K nearest neighbor regression]]

Un altro esempio di modello sono le [[Neural Networks|Reti neurali]], che sono in grado di approssimare relazioni non lineari tra input e output.
